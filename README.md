Overview
This is a internal project for Velir

Installation Guide
1. Run npm install to install all required npm packages
2. Run npm start to start server
3. To work with Mavenlink or Bamboo, you should create a config folder locally and put a txt file with your mavenlink or Bamboo api key or token.
4. You can use filehelper module to load your api keys. Just pass the whole filepath and get the key in callback function.
